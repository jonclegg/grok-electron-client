# Grok Electron Client

A native macOS desktop wrapper for Grok (xAI's conversational AI) built with Electron.

## Download

Download the latest release for your Mac:

- **Apple Silicon (M1/M2/M3/M4)**: [Grok Client-1.0.0-arm64.dmg](https://github.com/jonclegg/grok-electron-client/releases/download/v1.0.0/Grok.Client-1.0.0-arm64.dmg)

## Installation

1. Download the appropriate DMG file for your Mac
2. Open the DMG file
3. Drag "Grok Client" to your Applications folder
4. Launch from Applications

## Building from Source

1. Clone the repository:
   ```bash
   git clone git@github.com:jonclegg/grok-electron-client.git
   cd grok-electron-client
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run in development mode:
   ```bash
   npm start
   ```

4. Build for production:
   ```bash
   npm run build
   ```
